Basic HTML Tags



< a> This tag is used for one page to another.
< br> This tag is used for single line break.
< p> This tag is used for paragraph.
< small> This tag is used for making texts small.
< img> This tag is used for adding images to web page.
< div> This tag is used for creating a divison.
< font> This tag is used for changing font, color and size of a text.
< table> This tag is used for creating a table in a web page.
< tr> This tag is used for creating a table row.
< td> This tag is used for creating a table cell.
< u> This tag is used for underline the text.
< h1> < h2> < h3> This tags is used for changing the size of a text.
