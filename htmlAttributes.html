Tag Properties



This page contains usage of tag properties. Here, you can learn how to use modify tags.

In HTML you can modify your tag properties. Generally this is made in a CSS file but withou a CSS file, we can do these modifications.

For example:

< body background-color="chosen color" , align-items="chosen alignment style"


< h1 text-align: "chosen alignment style">


< font face="chosen text style" color="chosen color" size="chosen text size">


< a href="another page's link here" title="Text that clickable"> Text is here < /a>


Here you can see some examples for tag properties.
